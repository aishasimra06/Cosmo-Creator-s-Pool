# Creator Collaboration Hub

A modern full-stack collaboration platform that connects Developers, UI/UX Designers, Video Editors, and other creators to discover projects, build teams, and collaborate seamlessly within a unified workspace.

The platform streamlines the entire collaboration lifecycle—from project creation and applicant management to secure workspaces, real-time discussions, and project updates.

Features
🔐 Secure User Registration & Login (JWT Authentication)
👤 Creator Profiles & Portfolio Management
📁 Project Creation & Management
🔍 Explore and Search Collaboration Projects
📝 Apply for Projects with Personalized Requests
👥 Applicant Review & Recruitment Management
💬 Dedicated Collaboration Workspace
📢 Official Project Announcements
🔔 Notification System
📤 Media Upload & Portfolio Showcase
🛡 Role-Based Access Control
📱 Responsive Modern UI
